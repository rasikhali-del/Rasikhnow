# RASIKHNOW Requirements Document

## 1. Application Overview

### 1.1 Application Name
RASIKHNOW

### 1.2 Application Description
A fully responsive, modern Islamic website featuring Ramadan timetables for both Sunni and Shia communities, live analog clock, prayer time notifications, Ramadan gallery, and special duas. The website uses a dark Islamic theme with neon glow effects, glassmorphism design, and smooth animations.

### 2.4 Notification System
- Sahri and Iftaar alert notification system
- Request browser notification permission on first visit
- Automatically detect today's date and match with timetable
- Trigger notifications at:
  - Sehr time with message: Time for Sahri 🌙
  - Iftaar/Maghrib time with message: Time for Iftaar 🍽️
- Include sound alert if browser allows
- Do not trigger notification if time has already passed for the day
- Work daily without manual refresh
- Use JavaScript Notification API

- Smooth page transitions
